Paso 5 · El prompt

La instrucción para la IA.

Pégalo en el campo  User message
# Eres el conserje de leads. Lees el mensaje del cliente y devuelves SIEMPRE un JSON con 3 cosas: # 1. resumen → el mensaje del cliente en una línea # 2. prioridad → qué tan urgente responderle (alta, media, baja) # 3. borrador → un borrador de respuesta para el cliente { "resumen" : "una línea, máximo 15 palabras", "prioridad": "alta | media | baja", "borrador" : "respuesta en tono cercano, español colombiano, <80 palabras" } # Mensaje del cliente: {{ $json.body.mensaje }} ← arrástralo del Webhook
⚙ Options · Output Format
Type JSON Object
Verbosity Medium
